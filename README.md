# Disney API

Disney API is a Node.JS app, and consists of a REST API built upon web-scraped data.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) and [nodeJS](https://nodejs.org/en/) to install Disney API dependencies.

```bash
yarn install
```

## Usage

To start the app, use 

```bash
node index.js
```
## Documentation
Once the app is running, access the route http://localhost:1500/api-docs. The Swagger documentation includes tests of all the API routes as well.
